# MoveHero Motion Referee

MoveHero oyunu için hareket algılama ve test aracı.

## Nasıl Çalıştırılır?

Bu projenin telefon sensörlerine erişebilmesi için **HTTPS** üzerinden sunulması gerekir.

### Yöntem 1: Netlify Drop (En Kolay)
1.  Tarayıcıda [app.netlify.com/drop](https://app.netlify.com/drop) adresine gidin.
2.  `MoveHero_Referee` klasörünü (veya zip dosyasının içindekileri) sürükleyip bırakın.
3.  Size verilen Linki telefonunuzda açın.

### Yöntem 2: GitHub Pages
1.  GitHub'da yeni bir repository oluşturun (örn: `movehero-referee`).
2.  Bu klasördeki dosyaları (`index.html`, `style.css`, `referee.js`) oraya yükleyin.
3.  Repository ayarlarından (Settings > Pages) `Branch: main` seçip kaydedin.
4.  GitHub'ın verdiği linki telefonunuzda açın.
